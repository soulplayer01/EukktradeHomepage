# Broker Setup

## 브로커 연결 준비

- 거래를 위한 브로커 API 키를 발급받습니다.
- 브로커별 문서를 참고하여 계정과 API 권한을 확인하세요.

## API 키 관리

- API 키는 절대 공개 저장소에 커밋하지 마세요.
- 환경 변수 또는 시크릿 저장소를 사용하여 안전하게 관리합니다.

## 연결 테스트

- 브로커 연결이 가능한지 확인하려면 샘플 요청을 실행합니다.
- 응답이 정상적으로 오면 실제 자동매매 설정을 진행합니다.

## 브로커 구성 주의사항

- 브로커별 최대 주문 수량과 거래 수수료를 반드시 확인하세요.
- 시뮬레이션 환경과 실제 계정 환경을 혼동하지 않도록 주의합니다.
