# FAQ

## 자주 묻는 질문

### 1. 어디에서 개발 문서를 관리하나요?

`src/content/developers/` 아래의 Markdown 파일로 관리합니다. 추가 문서는 해당 폴더에 `.md` 파일로 추가하면 자동으로 반영됩니다.

### 2. 문서 내용은 어떻게 업데이트하나요?

로컬에서 Markdown 파일을 수정한 뒤 저장하면 개발 서버에서 즉시 반영됩니다. Vite의 HMR이 Markdown 파일을 감지합니다.

### 3. 기존 디자인 시스템 페이지는 유지되나요?

`/developers` 페이지는 공식 문서 페이지로 사용하며, 기존 `/design-system` 경로도 동일한 페이지로 유지됩니다.
