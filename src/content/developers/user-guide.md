# User Guide

## 기본 흐름

1. 로그인 또는 브로커 인증
2. 자동매매 전략 선택
3. 거래 조건 및 리스크 한도 설정
4. 백테스트 결과 확인
5. 자동매매 시작

## 주요 기능

- 전략 설정: 일목균형표 기반 전략을 선택하고 매수/매도 조건을 지정합니다.
- 상태 모니터링: 실행 중인 주문, 잔고, 손익 상태를 확인합니다.
- 로그 확인: 거래 내역과 전략 작동 기록을 검토합니다.

## 참고 사항

- 이 문서는 교육 및 개발용 안내자료이며 투자 권유가 아닙니다.
- 실제 투자 판단과 결과는 사용자 본인의 책임입니다.
